# 고급 커맨드 예시: /미팅준비

$ARGUMENTS 에 고객사 이름을 입력하면 미팅 준비 브리핑을 생성합니다.

## 순서

1. mock-data/sales/client-list.csv 에서 해당 고객사 정보 찾기
2. 최근 미팅 이력 확인
3. 계약 상태 확인
4. 미팅 브리핑 문서 생성

## 출력 형식

```
━━━ 미팅 브리핑: {고객사명} ━━━

■ 기본 정보
  담당자: {이름} {직책}
  계약: {금액} ({시작}~{종료})
  상태: {상태}

■ 최근 이력
  최근 미팅: {날짜}
  비고: {비고}

■ 준비 사항
  1. (자동 생성)
  2. (자동 생성)
  3. (자동 생성)
━━━━━━━━━━━━━━━━━━━━━━━━━━
```
