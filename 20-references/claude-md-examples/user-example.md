# CLAUDE.md 예시 -- 사용자용 (~/.claude/CLAUDE.md)

> 이 파일은 예시입니다. 사용자 레벨 CLAUDE.md는 모든 프로젝트에 적용됩니다.

---

## 나는 누구인가

- 이름: 홍길동
- 역할: PM (프로덕트 매니저)
- 회사: 스타트업 (직원 30명)
- 경력: PM 5년차

## 선호하는 커뮤니케이션

- 한글로 답변
- 결론부터 말하고 근거를 나중에
- 불렛 포인트 선호 (긴 문장 X)
- 전문 용어 사용 OK (PM 용어 알고 있음)

## 자주 하는 업무

1. PRD (제품 요구사항 문서) 작성
2. 스프린트 백로그 관리
3. 스테이크홀더 커뮤니케이션
4. 사용자 피드백 분석
5. 경쟁사 분석

## 주의사항

- "다 좋습니다" 같은 애매한 답변 하지 말 것
- 장단점을 항상 같이 제시할 것
- 숫자/데이터 기반 근거를 우선할 것
