## 프로젝트 개요

이 프로젝트는 마케팅팀의 주간 캠페인 성과 분석 자동화 시스템입니다.

## 나는 누구인가

- 역할: 마케팅팀 매니저
- 주요 업무: 광고 캠페인 관리, 성과 분석, 보고서 작성
- 사용 도구: Google Analytics, Meta Ads, 네이버 광고

## 규칙

- 보고서는 항상 표 형식으로 작성
- 숫자에는 천 단위 쉼표 표시
- 비교 시 전월 대비 증감률 포함
- 한글로 답변

## 자주 쓰는 패턴

### 캠페인 분석
```
입력: campaign-report.csv
출력: 매체별 성과 비교표 + 최적 매체 추천 + 개선 포인트 3가지
```

### 주간 보고서
```
매주 금요일 작성
포함 내용: 이번 주 성과 요약, 주요 지표 변화, 다음 주 액션 아이템
```
