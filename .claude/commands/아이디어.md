사용자가 떠올린 아이디어를 `40-ideas/`에 기록해줘.

## 기록할 아이디어

`$ARGUMENTS`에 있으면 그걸 쓰고, 없으면 "어떤 아이디어를 적을까?"라고 묻는다.

## 진행 방식

1. `40-ideas/` 에 `ideas.md`가 있으면 거기에 항목을 추가, 없으면 새로 만든다
2. 아이디어를 받으면 아래 형식으로 한 항목 추가한다:

   ```markdown
   ## (아이디어 제목)
   - **무엇**: 한 줄 설명
   - **왜**: 이게 있으면 뭐가 편한가
   - **종류**: 커맨드 / 스킬 / Hook / MCP 중 무엇에 가까운가
   - **난이도**: 쉬움 / 보통 / 어려움 (감으로)
   - **기록일**: (오늘 날짜)
   ```

3. 사용자가 막연하게 말하면 짧게 되물어 구체화한다 (질문 1~2개만)

## 규칙

- 기록이 목적이지 지금 당장 만들라는 게 아니다 — 가볍게 모아두기
- 추가 후, 현재 `40-ideas/`에 쌓인 아이디어 개수와 방금 추가한 항목을 알려준다
