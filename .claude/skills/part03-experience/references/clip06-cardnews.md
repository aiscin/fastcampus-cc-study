## 자동 셋업 (진행 안내 출력 직전 스킬이 즉시 Bash로 실행)

```bash
# 1) 워크스페이스 루트 자동 감지
WS="$(pwd)"
while [ "$WS" != "/" ] && [ ! -f "$WS/CLAUDE.md" ]; do WS="$(dirname "$WS")"; done
if [ "$WS" = "/" ]; then
  WS="$(pwd)"
  while [ "$WS" != "/" ] && [ ! -d "$WS/40-mock-data" ]; do WS="$(dirname "$WS")"; done
fi
[ "$WS" = "/" ] && { echo "⚠ 워크스페이스 루트 못 찾음"; exit 1; }
echo "워크스페이스: $WS"

# 2) 경로
DEST="$WS/50-my-work/Part03-체험하기/실습09-카드뉴스"
SRC="$WS/50-my-work/Part03-체험하기/실습08-자료리서치"

mkdir -p "$DEST"
cd "$DEST"

# 3) Clip 5 결과물(02-주제리서치.md) 통째 복사
if [ -d "$SRC" ]; then
  cp -r "$SRC"/* ./ 2>/dev/null
  rm -f README.md
fi

echo "=== 결과 ==="
ls -la
```

```
영상에서 STEP 1~5 순서대로 실습이 진행됩니다.
영상 일시정지·재생 자유롭게 하면서 본인 워크스페이스에서 같이 따라하세요.

■ STEP 흐름

  STEP 1. 작업 폴더 진입 (스킬이 폴더+02-주제리서치.md 자동 준비 완료)
    이미 셋업된 폴더: 50-my-work/Part03-체험하기/실습09-카드뉴스/

  STEP 2. 1단계 질문하기 (B) — "카드뉴스가 보통 어떻게?"
    "카드뉴스 만들고 싶거든.
     카드뉴스가 보통 어떤 형태·구조로 만들어지는지 알려줘.
     어떤 방법들이 있어?"
    클로드코드가 펼침: 플랫폼 규격(인스타 1080×1080 / 릴스 1080×1920 / 카톡 / 노션) +
              일반 구조(인트로·본문·정리·아웃트로) + 만드는 방법(기획서 먼저 → HTML).

  STEP 3. 2단계 기획하기 (U) — 본인 자료로 기획서 옵션
    "좋아. 그럼 이 02-주제리서치.md의 5개 이슈로 인스타 10장 카드뉴스 만들 건데,
     어떤 구조로 가면 좋을지 옵션 보여줘. 각 카드에 들어갈 내용 기획서까지."
    클로드코드가 구조 옵션 (A/B/C) + 각 카드 항목 (번호·제목·본문·시각·색상) 펼침.

  STEP 4. 3단계 만들기 (I) — 2단계 (기획서 → HTML)
    4-1: "그 구조로 01-카드뉴스-기획서.md 먼저 만들어줘."
    4-2: "기획서대로 02-카드뉴스.html 만들어줘.
          한 파일 안에 10장. 1080×1080 정사각, 세로 스크롤,
          Pretendard 폰트, Unsplash 이미지."

  STEP 5. 4+5단계 검토 + 부분 수정 (L+D)
    open 02-카드뉴스.html → 체크: 10장 채워짐? 한글 폰트? 가독성?
    부분 수정: "카드 3번만 배경색 #FFE5EC로. 다른 건 그대로"

■ 시작
1. 영상 시작 → STEP 1부터 따라가기
2. 영상 끝나면 이 채팅창에 "완료" 또는 "/wrap" 입력 → 결과물 자동 정리

※ 기획서 먼저 만드는 이유: 바로 HTML 가면 장수가 제멋대로(12장·20장)가 됨.
```
