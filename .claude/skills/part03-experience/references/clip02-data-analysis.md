## 자동 셋업 (진행 안내 출력 직전 스킬이 즉시 Bash로 실행)

> **이 단계는 명령 안내가 아니라, 스킬이 직접 Bash 도구로 실행한다.**
> 워크스페이스가 어디에 있든 자동 감지. 절대 경로 하드코딩 금지.

```bash
# 1) 워크스페이스 루트 자동 감지 (CLAUDE.md 위치)
WS="$(pwd)"
while [ "$WS" != "/" ] && [ ! -f "$WS/CLAUDE.md" ]; do
  WS="$(dirname "$WS")"
done
if [ "$WS" = "/" ]; then
  WS="$(pwd)"
  while [ "$WS" != "/" ] && [ ! -d "$WS/40-mock-data" ]; do
    WS="$(dirname "$WS")"
  done
fi
[ "$WS" = "/" ] && { echo "⚠ 워크스페이스 루트 못 찾음. CLAUDE.md 있는 폴더에서 claude 실행"; exit 1; }
echo "워크스페이스: $WS"

# 2) 작업 폴더 + 강의 자료 경로 (CLAUDE.md 표준)
DEST="$WS/50-my-work/Part03-체험하기/실습05-데이터분석"
MOCK="$WS/40-mock-data"

mkdir -p "$DEST"
cd "$DEST"

# 3) sample_csv 8개 CSV 가져오기 (Ch 02 시작이라 재료는 강의 자료에서)
if [ ! -d ./sample_csv ] && [ -d "$MOCK" ]; then
  mkdir -p sample_csv
  cp "$MOCK"/*.csv sample_csv/ 2>/dev/null
fi

echo "=== 결과 ==="
ls -la
```

**셋업 결과 한 줄 안내**:
- 성공 → "✓ `{WS}/50-my-work/Part03-체험하기/실습05-데이터분석/` 준비 완료. sample_csv/ 8개 CSV 가져왔어요. (쫀득팩토리 1분기 운영 데이터)"
- 40-mock-data 없음 → "⚠ 강의 자료(40-mock-data)를 못 찾았어요. CLAUDE.md 있는 워크스페이스에서 claude 실행했는지 확인해주세요."
