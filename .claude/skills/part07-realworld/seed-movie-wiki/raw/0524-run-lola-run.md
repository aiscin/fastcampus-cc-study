# Run Lola Run 감상 노트

[[Run Lola Run]]은 Lola가 남자친구 Manni를 살리기 위해 20분 안에 10만 마르크를 구해야 하는 1998년 독일 영화다. [[Tom Tykwer]]는 같은 시간을 세 번 반복해, 작은 선택과 우연이 전혀 다른 결과를 만든다는 구조를 액션처럼 밀어붙인다. [[Franka Potente]]의 빨간 머리와 베를린 거리 질주는 인물 심리보다 편집, 음악, 게임적 시간 감각을 먼저 남긴다.

## 핵심 정보

- Tom Tykwer가 각본과 연출을 맡았고 Franka Potente가 Lola, Moritz Bleibtreu가 Manni를 연기했다.
- Manni가 조직의 돈을 잃어버린 뒤, Lola는 아버지 은행, 거리의 사람들, 카지노, 구급차 같은 경로를 매번 다르게 통과한다.
- 영화는 라이브액션, 애니메이션, 플래시 포워드 사진 몽타주, 테크노 음악을 섞어 1990년대식 속도를 만든다.

## 주제·스타일

- 반복 구조는 운명론이 아니라 선택과 우연의 실험처럼 보인다. 같은 출발점도 리듬이 달라지면 결과가 달라진다.
- Lola의 달리기는 플롯 해결 수단이면서 영화 전체의 편집 원리다.
- [[The Lives of Others]]와 비교하면, 현대 독일 영화 안에서 이 작품은 침묵과 감시가 아니라 속도와 가능성의 영화다.

## 평가·수상

- 독일 영화상에서 주요 부문을 수상했고, 국제적으로 1990년대 독일 영화의 대표적 히트작으로 알려졌다.
- Sony Classics는 작품을 빠른 속도의 스릴러이자 반복 구조 영화로 소개한다.

## 연결

- [[Tom Tykwer]] - 음악과 편집, 시간 구조를 결합하는 감독.
- [[Franka Potente]] - Lola의 몸과 달리기가 영화의 아이콘이 된다.
- [[시간 영화]] - 반복되는 20분과 변주된 결과가 핵심 구조다.
- [[The Lives of Others]] - 통일 전후 독일 영화의 전혀 다른 속도와 정서를 비교하기 좋다.
