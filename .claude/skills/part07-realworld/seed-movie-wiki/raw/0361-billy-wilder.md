# Billy Wilder 웃다가 차갑게 베는 고전

[[Billy Wilder]]는 [[Some Like It Hot]]처럼 빠른 [[코미디]]를 만들다가도 [[Sunset Boulevard]]와 [[Double Indemnity]]처럼 할리우드와 욕망을 차갑게 들여다본다. 웃기고 로맨틱한 표면 아래에 계산, 거래, 자기기만이 남는 감독이라 [[필름 누아르]]와 [[로맨스]]가 한 파일 안에서 같이 열린다.

## 핵심 정보

- 오스트리아-헝가리 출생으로 베를린에서 각본 경력을 시작했고, 나치 집권 이후 할리우드로 이동해 감독·각본가·제작자로 활동했다.
- 대표작은 Double Indemnity, The Lost Weekend, Sunset Boulevard, [[Some Like It Hot]], [[The Apartment]], Witness for the Prosecution.
- Some Like It Hot은 금주법 시대 시카고에서 살인을 목격한 두 음악가가 여성 밴드로 변장해 도망치는 범죄 코미디다.
- The Apartment는 회사 승진을 위해 자신의 아파트를 상사들의 밀회 장소로 빌려주는 남자의 이야기로, 직장 코미디와 멜로드라마가 함께 간다.

## 주제·스타일

- Wilder의 농담은 친절하지 않다. 인물들이 원하는 것을 얻으려 할수록 도덕적 타협이 드러난다.
- Double Indemnity와 Sunset Boulevard에서는 욕망과 자기서사가 파멸을 부르고, Some Like It Hot과 The Apartment에서는 웃음이 사회적 위선을 찌른다.
- [[Alfred Hitchcock]]이 관객의 긴장을 조율한다면 Wilder는 대사와 상황으로 인물의 민낯을 드러내는 쪽이다.

## 평가·수상

- Wilder는 아카데미상을 여러 차례 받았고, The Apartment로 작품상·감독상·각본상 주요 부문을 휩쓸었다.
- Some Like It Hot은 개봉 당시와 회고 평가 모두에서 미국 코미디 영화의 대표작으로 반복해 언급된다.

## 연결

- [[Some Like It Hot]] — 범죄, 젠더 위장, 코미디 리듬이 한꺼번에 움직이는 대표작.
- [[The Apartment]] — 직장 윤리와 외로움을 다룬 로맨틱 코미디/드라마.
- [[Jack Lemmon]] — Wilder 코미디의 불안하고 인간적인 얼굴.
- [[필름 누아르]] — Double Indemnity와 Sunset Boulevard로 이어지는 어두운 고전 축.
