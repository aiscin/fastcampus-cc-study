# 01. PRD — TrendBoard 제품 정의

> 무엇을, 누구를 위해, 어떤 기능으로 만드는지. **클로드코드에게 가장 먼저 공유하는 문서.**
> show-me-the-prd가 생성할 PRD의 **시드/정답 레퍼런스**. (04 STEP 1에서 답할 내용, STEP 2 구체화의 기준)

---

## 1. 한 줄 요약

여러 유튜브 채널의 영상 성과를 **매시간 자동 수집**하여, **채널 평균 대비 얼마나 터졌는지(Outlier Score)** 와
다양한 지표로 **지금 뜨는 콘텐츠와 트렌드를 한눈에** 파악하는 대시보드.

---

## 2. 해결하는 문제

- 관심 채널이 많은데 새 영상을 일일이 확인하기 번거롭다
- 어떤 주제가 지금 뜨는지, 어떤 영상이 터졌는지 **감으로만** 판단한다
- 구독자 100만 채널과 1만 채널의 영상을 **같은 기준으로 비교**할 수 없다
- 영상을 하나하나 클릭해서 내용을 파악하는 데 시간이 든다

## 3. 핵심 가치

- **Outlier 감지** — 채널 평균 대비 성과 배율(Ch. Avg)로 "터진 영상"을 자동 식별
- **다각도 레이팅** — Outlier Score, V/S Ratio(구독자 대비 조회수), 인게이지먼트, 성장률
- **AI 요약** — 클릭 없이 한 줄 요약 + 키워드 + 콘텐츠 유형 태그로 내용 파악
- **트렌드 감지** — 반복 키워드와 급상승 토픽을 자동 추출해 랭킹

---

## 4. 사용자 & 시나리오

- **누구**: 본인 (특정 분야 콘텐츠를 추적하는 크리에이터/마케터/기획자)
- **상황**: 아침에 또는 콘텐츠 기획 시 대시보드를 열어 5분 안에 "지금 뭐가 뜨는지" 파악

**흐름:**
```
채널 등록 → 매시간 자동 수집 → Outliers에서 터진 영상 확인
→ AI 요약/키워드로 내용 훑기 → 레이팅 정렬로 비교 → Trends에서 뜨는 키워드 확인
```

---

## 5. 핵심 기능 (이번에 만드는 범위)

| 기능 | 설명 | 우선순위 |
|------|------|:---:|
| 채널 등록/관리 | URL 입력 → 자동 메타데이터 로드, 추가/삭제/활성토글, 카테고리 분류 | P1 |
| 매시간 자동 수집 | cron으로 최근 영상의 조회수/좋아요/댓글 스냅샷 저장 | P1 |
| **Outlier 대시보드** | Summary Cards + 영상 테이블(12컬럼) + 정렬/필터 — **앱의 핵심** | P1 |
| 다중 레이팅 | Score, Ch.Avg 배율, V/S Ratio, Views/Subs/Likes/Comments 정렬 | P1 |
| 포맷 구분 | Shorts vs Long 구분 + 필터 | P1 |
| AI 요약 + 태그 | Gemini로 한 줄 요약 + 키워드 + 콘텐츠 유형 태그 | P1 |
| 트렌드 키워드 랭킹 | 반복 키워드 추출 + 성장률 기반 핫토픽 | P1 |
| Overview | 전체 요약 카드 + Score 추이 차트 + 최근 Outlier | P1 |
| Admin 설정 | Gemini 키/모델/프롬프트 관리 (비밀번호 게이트) | P1 |
| AI 콘텐츠 전략 | 데이터 기반 전략/제목 공식 제안 | P2 (선택) |

---

## 6. 안 만드는 것 (Out of Scope)

> 클로드코드에게 이 목록도 함께 알려준다. 범위가 새는 걸 막는다.

- **로그인/회원가입** — 개인용. 내 URL로 직접 접속 (다중 사용자는 다음 단계)
- **결제/구독 플랜** — 개인용 단계에서 불필요
- **X(트위터)/Threads 연동** — YouTube가 가장 안정적·무료. 멀티플랫폼은 다음 단계
- **댓글 감성 분석** — 제목/설명 기반 요약까지만
- **모바일 네이티브 앱** — 반응형 웹으로 충분
- **자체 영상 재생** — 유튜브 링크로 이동

---

## 7. 성공 기준

- [ ] 채널 5개+ 등록 후 매시간 수집이 안정적으로 동작
- [ ] Ch. Avg 배율로 Outlier가 정확히 감지됨 (1.5x↑ = Outlier, 5x↑ = Mega)
- [ ] 다양한 기준으로 테이블 정렬/필터 가능
- [ ] 접속 5분 내 "이번 주 트렌드" 파악 가능
- [ ] AI 요약이 영상 내용을 80%+ 정확히 반영

---

## 8. 미리 정해둘 값 (기본값 제안)

| 항목 | 기본값 | 비고 |
|------|--------|------|
| 채널 최대 등록 | 10~20개 | YouTube 쿼터 고려 |
| Outlier 임계값 | Ch.Avg **1.5x↑** | Mega는 5.0x↑ |
| Mega Outlier | Ch.Avg **5.0x↑** | Top 0.1% |
| 채널당 수집 영상 | 최근 **10개** | playlistItems 1회 |
| 평균 조회수 기준 | 최근 **30일** | avg_views 계산 |
| 트렌드 집계 기간 | **7일 / 30일** 탭 | |
| 수집 주기 | **매시간** | pg_cron 또는 외부 cron |
| AI 요약 대상 | **요약 없는 신규 영상만** | 비용 절약 |
