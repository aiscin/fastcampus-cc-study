```
[STEP 1] 워크플로우 잡기
   "어떤 주제에 대해 딥리서치해서 정리하려는데, 어떻게 워크플로우를 구성해야 할까?"
   → 단계별 워크플로우 정리 (몇 단계가 나올지는 조사 결과에 따라)
[STEP 2] 보완 + 실행 가능 단계 정의
   정리된 단계들 살펴보고 보완 → 클로드코드가 실제로 할 수 있는 단계로 다듬기
[STEP 3] 스킬로 만들기
   정의된 흐름을 그대로 SKILL.md로 정리 (수강생 본인 v1)
[STEP 4] 만든 스킬로 테스트·검증
   "전국 김밥 맛집에 대해 깊이 리서치해줘" — description 매칭 + 결과 확인
[STEP 5] 본인 v1 한계 + clip-03 예고
```
