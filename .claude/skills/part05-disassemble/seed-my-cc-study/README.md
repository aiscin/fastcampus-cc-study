# 내 클로드코드 공부방 (my-cc-study)

> 강의 끝난 후, 혼자 클로드코드를 계속 공부하기 위한 워크스페이스

---

## 이 폴더는 뭐예요?

패스트캠퍼스 "클로드코드 뽀개기" 강의를 듣고 나서, **혼자서도 클로드코드를 계속 공부**하기 위한 폴더입니다.

강의 워크스페이스는 강사님이 만들어둔 거라서 잘 작동하지만, "내가 직접 처음부터 세팅한 워크스페이스"는 따로 있어야 진짜 본인 것이 됩니다.

이 폴더가 그 시작점이에요.

---

## 폴더 구조 (10단위 넘버링)

| 폴더 | 용도 |
|------|------|
| `10-curriculum/` | 강의 진행 노트 — Part 1~9 정리, 실습별 메모 |
| `20-concepts/` | **클로드코드 핵심 개념 정리** — CLAUDE.md / 커맨드 / MCP / Hook / 스킬 |
| `30-resources/` | 공식 문서 링크, 자주 쓰는 명령어 치트시트, 영감 받은 사례 |
| `40-ideas/` | 본인이 만들고 싶은 자동화/스킬 아이디어, 더 파볼 기능 목록 |
| `sandbox/` | 실험용 — 뭐든 시도해보고 안 되면 지우기 |

## 다음 할 일

이 폴더에 `CLAUDE.md` 파일이 **아직 없어요**.

Part 05 강의에서 직접 만들게 됩니다:

1. 이 폴더에서 `claude` 실행
2. `/init` 명령으로 클로드코드가 폴더 분석 → CLAUDE.md 초안 생성
3. 인터뷰로 본인 맞게 다듬기

---

## 학습 흐름 추천

1. **모르는 개념 만나면** → `20-concepts/` 해당 노트 먼저 보기
2. **공식 답 필요하면** → `30-resources/공식문서-링크.md` 따라가기
3. **남이 만든 거 보고 영감받으면** → `30-resources/영감-받은-사례.md`나 `40-ideas/`에 적기
4. **실험하고 싶으면** → `sandbox/`에서 자유롭게
5. **결과 정리되면** → 새 노트로 `20-concepts/`에 추가

## 강의 워크스페이스와의 차이

| 강의 워크스페이스 (`~/fastcampus-claudecode/workspaces/fastcampus-cc/`) | 이 워크스페이스 (`~/fastcampus-cc/50-my-work/Part05-뜯어보기/my-cc-study/`) |
|---|---|
| 강사가 미리 세팅한 것 | 본인이 처음부터 만드는 것 |
| 절대 수정 금지 (실습 진행 도구) | 자유 수정 OK |
| 실습 안내 스킬(`/part02` 등) 내장 | 본인 노트/아이디어 정리용 |
