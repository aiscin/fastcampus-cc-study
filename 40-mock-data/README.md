# 쫀득 팩토리 — 데이터 소개

> 이 데이터는 가상의 프리미엄 디저트 제조 회사 **"쫀득 팩토리"**의 2026년 1~3월 공장 운영 데이터입니다.
> 실습에서 이 데이터를 분석하고, 보고서를 만들고, 대시보드를 만들어봅니다.

---

## 회사 소개

**쫀득 팩토리**는 프리미엄 디저트를 만드는 제조 회사입니다.

- 2개 생산 라인: **DUBAI**(두바이 라인), **ITALY**(이태리 라인)
- 4가지 제품 생산
- 10종 이상의 원재료 사용
- 2교대 운영 (A조/B조)

### 제품 라인업

| 라인 | 제품코드 | 제품명 | 판매가 |
|------|---------|--------|--------|
| DUBAI | DCK-C | 두바이코코아 | 6,000원 |
| DUBAI | DCK-M | 두바이말차 | 6,000원 |
| ITALY | ICK-O | 이태리오리지널 | 6,000원 |
| ITALY | ICK-D | 이태리다크 | 6,000원 |

## 데이터 구성

### CSV 파일 (AI 분석용)

| 파일 | 내용 | 행수 | 주요 컬럼 |
|------|------|------|----------|
| `production.csv` | 일별 생산 실적 | 252행 | 생산일자, 라인, 제품명, 계획수량, 생산수량, 양품수량, 불량수량, 달성률 |
| `defect.csv` | 불량 상세 기록 | 1,306행 | 불량일자, 제품명, 불량유형, 원인, 조치사항, 처리구분 |
| `equipment.csv` | 설비 가동 기록 | 504행 | 기록일자, 설비명, 계획가동시간, 실가동시간, 비가동사유, 라인효율 |
| `material_io.csv` | 원재료 입출고 | 934행 | 일자, 구분(IN/OUT), 원재료명, 거래처, 수량, 단가, 금액 |
| `inventory.csv` | 원재료 재고 현황 | 168행 | 스냅샷일자, 원재료명, 현재고, 안전재고, 재고상태, 재고일수 |
| `bom_detail.csv` | 제품별 원재료 투입 (BOM) | 36행 | 제품명, 원재료명, 개당투입량, 단가, 개당원가 |
| `product_cost.csv` | 월별 제품 원가 | 12행 | 월, 제품명, 판매가, 재료비, 총원가, 이익률 |
| `kpi_monthly.csv` | 월별 KPI 요약 | 3행 | 월, 생산달성률, 불량률, OEE, 가동률, 이익률 |

## 주요 이벤트

| 월 | 상황 |
|----|------|
| 1월 | 정상 운영. 생산달성률 96.9%, 불량률 1.64% |
| 2월 | DUBAI 라인 4일 중단 + 피스타치오 단가 급등. 달성률 85%로 하락 |
| 3월 | 생산 회복(97%). 계절 품질 이슈 발생, 불량률 2.16% |

## 실습에서 이 데이터로 할 수 있는 것

- **데이터 분석** — 라인별 생산 효율 비교, 불량 원인 파악, 재고 위험 품목 식별
- **보고서 작성** — 분석 결과를 경영진용 보고서로 정리
- **대시보드 만들기** — 생산/불량/재고를 한눈에 보는 인터랙티브 대시보드
