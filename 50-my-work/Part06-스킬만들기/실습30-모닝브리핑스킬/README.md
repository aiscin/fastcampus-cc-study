## 무엇을 만들었나

### 1) daily-news 스킬 (뉴스 칸 재료)
- 네이버 검색 API로 정치·경제·사회·IT·세계·속보 **분야별 다중 검색**
- 같은 사건을 보도한 **매체 수(보도량)로 중요도 랭킹** → 상위 5건
- **연예·화보성 기사 필터** (여러 매체가 같은 행사를 찍어 보도량이 부풀려지는 오탐 방지)
- 스크립트: `scripts/fetch_daily_news.py` (수집·군집·랭킹·필터 = 결정론)

### 2) morning-briefing 스킬 (3+소스 통합)
아침 브리핑 구성:
```
🌤️ 날씨+옷차림  →  📅 오늘 일정(월요일이면 주간)  →  ⚠️ 겹침 알림
→  📧 답장 우선순위 TOP3  →  📆 메일→일정 등록 제안  →  📰 뉴스 5선
```
- 📧 **메일** — Gmail 안 읽은 총 개수 + 최근 10건, **광고 필터**(단, 보안·결제·배송 발신은 보존 → 오탐 방지)
- 📅 **일정** — 오늘 일정 / **월요일엔 주간 일정** / **겹치는 일정 자동 감지**
- 🌤️ **날씨** — Open-Meteo 무료 API (키 불필요) + 옷차림 한 줄
- 📆 **메일→일정 제안** — 메일에서 날짜·미팅 신호 감지 시 캘린더 등록 제안
- 📰 **뉴스** — daily-news 스킬 재호출
- 출력: 대화로 짧게 브리핑 + `brief-{날짜}.md` 저장 (멱등)
- 스크립트: `scripts/gather.py` (메일·일정·날씨 수집 + 광고 필터 + 겹침 감지 = 결정론)
