# daily-news — 오늘의 종합 뉴스 헤드라인

> 분야별 다중 검색 → 보도량 랭킹으로 "오늘 제일 중요한 뉴스"를 근사한다.
> 결정론 영역(수집·군집·랭킹·연예필터)은 `scripts/fetch_daily_news.py`가 처리하고,
> 이 문서(AI)는 **최종 선별·요약·톤 조정**만 맡는다.

> ⚠️ 공개 뉴스를 AI로 요약한 정보 제공용이다. 정확성을 보장하지 않으니 중요한 사안은 원문 확인을 권한다.

---

## 왜 이렇게 만드나

네이버 검색 API엔 "헤드라인" 엔드포인트가 없다. 그래서 분야별 대표어로 여러 번 검색한 뒤,
**같은 사건을 여러 매체가 보도할수록 그날 중요한 뉴스**라는 원리로 보도량을 세어 순위를 매긴다.
이 계산(수집·유사도 군집·랭킹)은 매번 같아야 하므로 스크립트로 고정했다. AI는 뽑힌 후보에서
맥락상 덜 중요한 것을 걷어내고 사람이 읽기 좋게 요약하는 판단만 한다.

## 사전 준비 (처음 한 번만)

네이버 검색 API 키가 프로젝트 루트 `.env`에 있어야 한다 (naver-news 스킬과 공용):
```
NAVER_CLIENT_ID=발급받은_ID
NAVER_CLIENT_SECRET=발급받은_Secret
```
키 발급은 [developers.naver.com](https://developers.naver.com) → 애플리케이션 등록 → 검색 API.

## 실행 흐름

### STEP 1 — 뉴스 수집·랭킹 (결정론, 스크립트)

```bash
cd <프로젝트 루트>
set -a; source .env; set +a
python3 .claude/skills/daily-news/scripts/fetch_daily_news.py --top 5
```

옵션: `--top N`(반환 수, 기본 5), `--per-cat M`(한 분야 독식 방지 상한, 기본 2).

출력 JSON: `total_collected, clusters, errors[], has_error, top[]`.
각 top 항목: `title, coverage(보도 매체 수), cats[](분야), desc, link`.

**에러 처리** — `has_error: true`이면 (키 오류·호출한도 등) 발행하지 말고 `errors[]`를 한국어로 안내한다.
일부 분야만 실패(`errors`는 있으나 `top`은 채워짐)면 "일부 분야 수집 실패" 한 줄 붙이고 진행한다.

### STEP 2 — 최종 선별·요약 (AI 판단 영역)

스크립트가 뽑은 `top`을 읽고:
1. **맥락 필터** — 스포츠 이적설·지역 행정 등 "전국적 중요도가 낮은" 항목이 섞였으면 뒤로 미루거나 제외한다. `coverage`가 높은 것(여러 매체 보도)을 우선한다.
2. **한 줄 요약** — 각 뉴스를 사실 위주 1~2문장으로. `desc`를 근거로 하되 단정·추측은 피한다.
3. **분야 균형** — 한 분야(예: 정치)만 5건이 되지 않게 섞는다.
4. 뉴스가 부족하면(수집 실패·전부 저품질) 억지로 5건 채우지 말고 있는 만큼만 정직하게 보여준다.

### STEP 3 — 출력

대화창에 **번호 매긴 5선**으로 보여준다. 형식:
```
📰 오늘 알아야 할 뉴스 (YYYY-MM-DD)
1. [제목 핵심] — 한 줄 요약  (N개 매체 보도)
...
ℹ️ 공개 뉴스 AI 요약 · 정보 제공용
```
morning-briefing이 호출한 경우엔 이 5선을 그 스킬의 뉴스 칸 재료로 넘긴다(별도 저장 없음).
단독 호출이면 대화 설명으로 끝낸다(파일 저장은 사용자가 요청할 때만).

## Settings (가변 요소)

| 설정 | 기본값 | 변경 방법 |
|------|--------|-----------|
| 뉴스 개수 | 5 | `--top N` |
| 분야당 상한 | 2 | `--per-cat M` |
| 검색 분야 | 정치·경제·사회·IT/과학·세계·속보 | `scripts/fetch_daily_news.py`의 `CATEGORIES` 수정 |
| 연예·화보 필터 | 켜짐 | 같은 파일 `FLUFF_PATTERNS` 수정 |

## Scripts
- **`scripts/fetch_daily_news.py`** — 분야별 다중 검색·자카드 유사도 군집·보도량 랭킹·연예/화보 필터·분야 균형. `.env`의 네이버 키 사용.
