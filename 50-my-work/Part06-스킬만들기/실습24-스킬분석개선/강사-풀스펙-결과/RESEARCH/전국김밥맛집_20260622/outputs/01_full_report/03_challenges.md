# 3. 검증의 한계와 신뢰도 이슈

## 3.1 출처 신뢰도 구조

본 리서치의 정보는 출처 등급별로 신뢰도가 다르다.

| 영역 | 주 출처 등급 | 신뢰도 |
|---|---|---|
| 김밥 역사·유래(충무김밥·국풍81) | B (한국민족문화대백과, 국가유산청) | 높음 (본문 교차검증) |
| 개별 가게 존재·시그니처 | C (여행 매체·다이닝코드·식신) | 중간 |
| 영업시간·가격·평점 | C (플랫폼, 변동성 큼) | 낮음~중간 (재확인 필요) |
| '원조' 주장 | C~E (가게/매체 주장) | 낮음 (검증 곤란) |

## 3.2 핵심 한계 4가지

1. **'원조' 단정 불가**: 광장시장 마약김밥, 통영 충무김밥 모두 여러 가게가 경쟁적으로 '원조'를 주장한다. 본 보고서는 "원조로 평가받는다" 수준으로만 기술.
2. **영업정보 변동성**: 시간·가격·휴무는 수시로 바뀐다. 특히 오월의김밥(조기 마감)·다가미(포장 전용)·김밥신화(예약 필수)는 방문 전 재확인 필수.
3. **플랫폼 평점의 한계**: 다이닝코드·식신 평점은 표본·시점에 따라 흔들린다. 절대 순위로 받아들이지 말 것.
4. **권역 커버리지 불균형**: 서울·통영·제주는 자료가 풍부하나, 속초·광주·충북 등은 신뢰할 개별 정보가 부족해 얕게 다뤘다(추후 보강 대상).

## 3.3 교차검증 기록 (요지)

- 충무김밥 "국풍81 전국화" → 한국민족문화대백과 본문으로 **직접 확인**(2출처 이상 정합).
- 마약김밥 별명 유래·모녀김밥 원조 평가 → 트립닷컴·다이닝코드·비짓서울 복수 매체 정합, 단 '원조'는 단정 회피.
- 강릉·대구 가게는 다이닝코드 평점 1차 의존 → "플랫폼 평점" 명시.
