# 50-my-work — 내가 만든 결과물

이 폴더는 비어있습니다. **강의 진행하면서 자동으로 채워집니다.**

---

## 자동 저장 구조

각 실습 결과는 다음 경로에 자동 저장됩니다:

```
50-my-work/
└── Part{NN}-{제목}/
    └── 실습{NN}-{제목}/
        ├── README.md          ← 무엇을 만들었는지 + 날짜·모드·모델
        └── (결과물 파일들)
```

**예시**:
```
50-my-work/
├── Part02-시작하기/
│   ├── 실습03-첫실행-폴더정리/
│   └── 실습04-모드소개-Alias/
├── Part03-체험하기/
│   ├── 실습05-데이터분석/
│   │   ├── README.md
│   │   ├── analysis.md
│   │   └── insights.csv
│   └── ...
└── Part06-스킬만들기/
    └── 실습23-딥리서치스킬/
        ├── README.md
        └── skill-source/      ← 만든 스킬 사본 (.claude/skills 활성화 + 여기 백업)
```

---

## 강의 가이드 스킬이 자동 처리

`/part02`, `/part03` 등 가이드 스킬 호출 시:
1. 해당 Part 폴더 자동 생성 (없으면)
2. 실습별 폴더 자동 생성
3. 결과물 + 메타 정보 README 함께 저장
4. `progress.json`에 완료 기록

학생은 폴더 신경 안 쓰고 강의만 따라가시면 됩니다.

---

## 진도/레벨 확인

```
/my-progress    ← 현재까지 완료한 실습 목록
/check-level    ← 지금 레벨 (AI Starter ~ AI Native)
/막혔어요       ← 에러/문제 해결 가이드
```

---

## 본인이 만든 자산 (Part 04~06)

직접 만든 스킬·커맨드·hook은:
- **활성화 위치**: `.claude/skills/`, `.claude/commands/`, `.claude/hooks/` (클로드코드가 자동 인식)
- **포트폴리오 사본**: `50-my-work/Part{NN}-.../skill-source/` 등 (어떤 실습에서 만들었는지 추적용)
