# 딥리서치 쿼리 프로파일 (반복 리서치용)

deep-research 스킬은 JSON 구조화 쿼리를 받으면 **Phase 1(인터뷰 4문항)을 통째로 건너뛴다.**
자주 하는 리서치 유형을 프로파일로 저장해 두고, 대문자 자리표시자만 바꿔 넣으면 인터뷰 없이 바로 시작한다.

## 3종 프로파일

| 파일 | 언제 | 바꿔 넣을 자리 |
|---|---|---|
| `시장동향.json` | ○○ 시장 규모·성장·전망을 정기 추적 | `TOPIC`, `keywords` |
| `기술비교.json` | A vs B 도구·기술 선택 판단 | `OPTION_A`, `OPTION_B`, `keywords` |
| `이슈추적.json` | 사건·이슈의 원인·경과·영향 파악 | `ISSUE`, `keywords` |

## 쓰는 법

1. 프로파일 파일을 복사한다.
2. 대문자 자리표시자(`TOPIC` 등)를 실제 주제로 치환한다.
3. 클로드코드에 이렇게 말한다:
   > "이 프로파일로 딥리서치 하려는데 — (파일 경로 첨부/붙여넣기)"

기본 출력은 **executive_brief(요약 중심)**로 맞춰 뒀다 — 반복 용도엔 20~50페이지 종합보고서가 과하기 때문.
종합보고서가 필요하면 `output.format`을 `comprehensive_report`로, `max_words`를 늘린다.

> 새 유형이 반복되면 프로파일을 하나 더 추가한다. 스키마: deep-research 플러그인의 `references/query_schema.json`.
