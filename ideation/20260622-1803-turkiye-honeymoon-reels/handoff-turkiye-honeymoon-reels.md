# 핸드오프 — 튀르키예 신혼여행 릴스 대본기 (개인화 파이프라인)

생성: 2026-06-22 18:03 · ideation-funnel
실행 폴더: `ideation/20260622-1803-turkiye-honeymoon-reels/`

## 개요
- **정의**: 여행지(예: 튀르키예 도시)를 입력하면 → 내 인스타 계정(@travel_yonori) 톤을 분석하고 → 잘되는 정보형 여행 릴스 템플릿을 입혀 → deep-research로 모은 꿀팁을 → 9:16 인스타 릴스 대본(씬별 표)으로 뽑아주는 스킬.
- **왜**: 신혼여행 영상·사진은 쌓이는데 릴스 대본 짜기가 막막함. 게다가 "내 계정 색"을 유지하면서 검증된 릴스 공식까지 입히고 싶음 → 수작업으로는 매번 어려우니 자동화.
- **누구**: 본인 (신혼여행 브이로그를 @travel_yonori에 올리는 개인)

## 레퍼런스 정리
- 릴스 스크립트 작성 스킬: **기존에 없음** → 새로 만들 가치 있음
- **재활용 자산**: `deep-research` 스킬 — 여행지 맛집·동선·꿀팁 수집 단계에서 호출
- 차별점: 단순 대본 생성이 아니라 **내 계정 톤 학습 → 잘되는 타 계정 구조 결합** 개인화

> **★ 전문가 팀 검토(Step 4.5) 반영분** — 아래는 마케터·카피라이터·영상기획자·비평가 검토 후 채택한 개선이 들어간 버전이다. (검토 원본: `resource/expert-review-2026-06-22.md`)

## 핵심 기능 (우선순위)
1. ★MVP **내 계정 톤 분석 (캡션 붙여넣기 = 주 입력)** — WebFetch 인스타 크롤링은 로그인월로 불확실하므로, **사용자가 본인 릴스 캡션 9~12개를 붙여넣으면** 말투·이모지·해시태그 패턴을 분석해 "내 톤 프로필" 작성. (WebFetch는 되면 보조, 안 되면 캡션이 기본 — 비평가·마케터 합의)
2. ★MVP **잘되는 타 계정 템플릿 도입** — 정보형 여행 릴스 성과 좋은 구성 템플릿(훅 공식·씬 구성·자막 리듬)을 **구조만** 참고해 내 톤과 결합. 표현 복제 금지(표절·클론 낙인 방지).
3. ★MVP **정보 수집 + 신선도 경고** — deep-research로 꿀팁·맛집·동선 수집. **가격·영업시간 포함 씬엔 "██ 기준, 현지 확인 필수" 자동 삽입**(비평가 방어장치).
4. ★MVP **대본 생성 (강화된 씬별 표 + 훅 2안)** — 맞춤 템플릿 + 정보로 9:16 대본 작성. **훅은 2안(숫자형/질문형) 생성 후 사용자가 선택** → 선택안을 씬1에 반영.
5. ★MVP **해시태그 + CTA 3종** — 해시태그 3~5개(정밀) + **릴스 목적별 CTA**(저장형/팔로우형/댓글형) 중 골라 표 마지막 행에 삽입.
6. (v2) 영상 길이별 씬 수 자동분기 / 기촬영 영상 모드 / 샷리스트 / 시리즈 / 톤 프로필 캐싱.

## 입력 / 출력
- **입력**: 여행지·도시명 (필수) + **내 릴스 캡션 9~12개 붙여넣기**(톤 분석용, 주 입력) + (선택) 영상 장면·일정 메모
- **출력**: 마크다운 **강화 씬별 표**
  - 컬럼: `씬# | 시작~끝(초) | 샷타입+화면 | 자막(블록) | 내레이션`
    - 샷타입 약어 첫 토큰 강제: `[WS]/[MS]/[CU]/[POV]/[트래킹]` (예: `[WS] 그랜드바자르 입구 정면`)
  - 표 아래 **업로드 패키지 블록**: `해시태그(3~5)` + `CTA(목적 선택)` + `게시 캡션`
  - 상단: "내 톤 프로필" 요약 1~2줄 + 선택한 훅

## 구현 단계 (클로드코드가 따라갈 흐름)
1. **톤 분석** — 사용자에게 "최근 릴스 캡션 9~12개 붙여넣어 주세요" 요청 → 말투·이모지·해시태그 패턴 파싱 → 톤 프로필. (WebFetch @travel_yonori는 가능하면 보조 시도, 막히면 캡션만으로 진행)
2. **템플릿 결합** — 정보형 릴스 잘되는 구성(resource 리서치)을 구조만 가져와 톤 프로필과 결합 → 맞춤 템플릿.
3. **정보 수집** — `deep-research` 호출, 출처 검증분만. 가격·시간 정보엔 신선도 경고 태깅.
4. **훅 2안 → 선택** — 숫자형/질문형 훅 2개 제시, 사용자 선택.
5. **대본 작성** — 맞춤 템플릿에 정보 채워 강화 씬별 표(샷타입·타임코드 포함) 생성. 자막 씬당 1줄·중앙 배치 전제.
6. **마무리** — 해시태그 + 목적별 CTA + 캡션 = 업로드 패키지로 묶어 출력·저장.

```
열기구·동굴호텔·포토스팟 꿀팁 넣고, 씬별 표 + 해시태그 + 캡션까지.
```

## 수용 기준 (이게 되면 완성)
- [ ] 캡션 9~12개 붙여넣기만으로 "내 톤 프로필"이 추출된다 (인스타 크롤링 의존 X)
- [ ] 강화 씬별 표가 나온다 (샷타입 약어 + 시작~끝 초 + 자막 컬럼 채워짐)
- [ ] 훅이 2안 제시되고, 고른 안이 씬1에 반영된다
- [ ] 가격·영업시간 정보엔 "기준 시점 + 현지 확인" 경고가 붙는다
- [ ] 해시태그 3~5개 + 목적별 CTA + 캡션이 업로드 패키지로 함께 출력된다
- [ ] 정보는 deep-research 검증분에서 나오고, 출처 불명확한 단정이 없다
